fun LinearGradient(limits: Pair<Color, Color>)

Returns the linear gradient from the first color of limits with offset 0 to the second color of limits with offset 1. coordinate parameters are default.

fun LinearGradient(colors: List<Color>)

Returns the linear gradient based on the colors in the list. Offsets are calculated by the position of the color in the list.

fun LinearGradient(x: Double, y: Double, x2: Double, y2: Double, colors: List<Color>)

Returns the linear gradient with first four parameters x, y, x2, y2, ranging from 0 - 1, corresponding to the percentage in the graphical wraparound box.
