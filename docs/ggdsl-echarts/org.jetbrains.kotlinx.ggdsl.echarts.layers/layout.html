layout

inline fun LayerPlotContext.layout(block: EChartsLayout.() -> Unit)

Adds a layer with settings and features to plot.

  • size - plot size.

  • title - org.jetbrains.kotlinx.ggdsl.echarts.features.Title of plot.

  • textStyle - org.jetbrains.kotlinx.ggdsl.echarts.features.TextStyle.

  • grid - grid settings.

  • legend - legend.

  • tooltip - tooltip.

plot {
    layout {
        size = 500 to 400
        title {
            text = "Main Title"
            subtext = "Subtitle"
        }
        legend {}
        tooltip { trigger = Trigger.AXIS }
        grid { left = 60.px; top = 10.pct; right = 60.px; bottom = 10.pct }
        textStyle.color = Color.RED
    }
}
