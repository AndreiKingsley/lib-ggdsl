inline fun LayerCollectorContextImmutable.bars(block: BarContextImmutable.() -> Unit)

Adds a new bars layer.

Creates a context in which you can create bindings using aesthetic attribute properties invocation and add features.

  • x - mapping data on the x-axis.

  • y - mapping data on the y-axis.

  • color - bars color.

  • alpha - bars opacity.

bars {
   x(categories)
   y(values)
   color(Color.LIGHT_GREEN)
   alpha(.5)
}
