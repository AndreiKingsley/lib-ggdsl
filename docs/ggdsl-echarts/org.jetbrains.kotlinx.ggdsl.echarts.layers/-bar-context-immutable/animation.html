animation

val animation: Animation

Animation options settings for bars. If a property isn't set or set to null, a default value will be used.

  • enable - responsible for enabling animation. By default true.

  • threshold - sets a graphic number threshold for animation. Animation will be disabled when graphic number exceeds a threshold. By default 2000.

  • duration - duration of the first animation. By default 1000.

  • easing - easing effect used for the first animation. By default cubicOut.

  • delay - delay before updating the first animation. By default 0.

plot {
 bars {
     animation {
         enable = true
         threshold = 2000
         duration = 1000
         easing = AnimationEasing.CUBIC_OUT
         delay = 0
     }
 }
}
