legend

inline fun EChartsLayout.legend(crossinline block: Legend.() -> Unit)

Shows name, symbol and color of different layers.

  • type - type of legend. plain is default.

  • left - distance between a legend component and the left side of the container. auto by default.

  • top - distance between a legend component and the top side of the container. auto by default.

  • right - distance between a legend component and the right side of the container. auto by default.

  • bottom - distance between a legend component and the bottom side of the container. auto by default.

  • width - width of a legend component. auto by default.

  • height - height of a legend component. auto by default.

  • orient - the layout orientation of legend. horizontal by default.

  • formatter - formatter is used to format label of legend.

plot(mapOf()) {
    layout {
        legend {
            type = LegendType.PLAIN
            left = 10.pct
            top = 60.px
            right = 10.pct
            bottom = 60.px
            width = 300.px
            height = 10.pct
            orient = Orient.VERTICAL
            formatter = "Legend {name}"
        }
    }
}
