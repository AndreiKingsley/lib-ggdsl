facet

fun facet(variable: ColumnReference<*>, order: OrderDirection = OrderDirection.ASCENDING, format: String? = null)

Adds a a new facet by a given variable.

Parameters

variable

Variable which defines this facet.

order

Specifies ordering direction of this facet.

format

Specifies the format pattern for displaying faceting values in rows.

Format pattern in the format parameters can be just a number format (like "d") or a string template where number format is surrounded by curly braces: "{d} cylinders". Note: the "$" must be escaped as "\$"

Examples: ".2f" -> "12.45" "Score: {.2f}" -> "Score: 12.45" "'Score: {}' "-> "Score: 12.454789"
