inline fun LayerCollectorContextMutable.abLine(block: ABLineContextMutable.() -> Unit)

Adds a new line with specified slope and intercept layer.

Creates a context in which you can create bindings using aesthetic attribute properties invocation. In this context, you can use mutable mappings - that is, do mapping and scaling with iterables.

Aesthetic attributes:

Positional:

  • slope - line slope.

  • intercept - line intercept.

Non-positional:

  • color - line color, of the type Color, mappable. (TODO grouping)

  • alpha - layer alpha, of the type Double, mappable. (TODO grouping)

  • type - line type, of the type LineType, mappable. (TODO grouping)

  • width - line width, of the type Double, mappable. (TODO grouping)

abLine {
   slope(0.5) // setting of constant `slope` value
   color(listOf("A", "B", "B", "A").scaled(..)) // mapping from list to `color` with some scale
}
