inline fun LayerContextInterface.tooltips(columns: List<ColumnReference<*>> = listOf(), title: String? = null, anchor: Anchor? = null, minWidth: Double? = null, hide: Boolean = false, columnsFormats: Map<ColumnReference<*>, String> = mapOf(), aesFormats: Map<Aes, String> = mapOf(), statFormats: Map<Statistic<*>, String> = mapOf(), tooltipsContextAction: LayerTooltipsContext.() -> Unit)

Defines the format for displaying values of this layer.

Creates a LayerTooltipsContext. In this context, you can configure lines of tooltip by using line(..) methods.

See also

LayerTooltipsContext

value

Parameters

columns

list of ColumnReference to crete a general multiline tooltip with. Useful for specifying the tooltip content quickly, instead of configuring it via the line(..) methods.

title

the string template to use as a title in the multi-line tooltip.

anchor

the fixed position for the general tooltip.

minWidth

minimum width of a general tooltip in pixels.

hide

flag of tooltips displaying.

columnsFormats

map of ColumnReference to format string of its value.

aesFormats

map of Aes to format string of its value. The format will be applied to the mapped value in the default tooltip or to the corresponding value specified in the line template.
