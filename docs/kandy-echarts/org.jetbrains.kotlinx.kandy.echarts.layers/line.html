inline fun LayerCollectorContext.line(block: LineContext.() -> Unit)

Adds a new line layer.

Creates a context in which you can create bindings using aesthetic attribute properties invocation and add features.

  • x - mapping data on the x-axis.

  • y - mapping data on the y-axis.

  • color - line color.

  • symbol - symbol on line. Symbols are not shown by default.

  • smooth - smooth curve. false by default.

  • alpha - line opacity.

  • width - line width. 2 by default.

  • lineType - line type. solid by default.

  • step - step line. false by default.

  • cap - end points of line. butt by default.

  • shadowColor - shadow color of line.

  • shadowBlur - shadow blur size of line.

line {
   x(time)
   y(values)
   color = Color.RED
   symbol = Symbol.CIRCLE
   smooth = true
   alpha = .9
   width = 2.0
   lineType = LineType.SOLID
   step = Step.FALSE
   cap = Cap.BUTT
   shadowColor = Color.GREY
   shadowBlur = 10
}
