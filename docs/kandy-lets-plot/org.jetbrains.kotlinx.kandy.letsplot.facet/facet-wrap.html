facetWrap

fun LayerPlotContext.facetWrap(nCol: Int? = null, nRow: Int? = null, scalesSharing: ScalesSharing = ScalesSharing.FIXED, direction: Direction = Direction.HORIZONTAL, block: FacetWrapContext.() -> Unit)

Splits data by one or more faceting variables. For each data subset creates a plot panel and lays out panels according to the nCol, nRow and direction settings.

Opens a FacetWrapContext. FacetWrapContext.facets is defined in this context. This method adds a new facet by a given variable.

facetWrap(nRow = 3, scalesSharing = ScalesSharing.FREE_X) {
   facet(col1)
   facet(col2, OrderDirection.DESCENDING)
   facet(col3, format = {.2f})
}
