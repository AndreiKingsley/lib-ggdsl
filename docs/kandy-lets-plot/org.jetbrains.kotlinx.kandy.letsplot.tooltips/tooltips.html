inline fun LayerContext.tooltips(variables: List<ColumnReference<*>> = listOf(), title: String? = null, anchor: Anchor? = null, minWidth: Double? = null, hide: Boolean = false, formats: Map<ColumnReference<*>, String> = mapOf(), tooltipsContextAction: LayerTooltipsContext.() -> Unit = {})

Defines the tooltips format for this layer.

Creates a LayerTooltipsContext. In this context, you can configure lines of tooltip by using line(..) methods.

Parameters

variables

list of columns to crete a general multiline tooltip with. Useful for specifying the tooltip content quickly, instead of configuring it via the line(..) methods.

title

the string template to use as a title in the multi-line tooltip.

anchor

the fixed position for the general tooltip.

minWidth

minimum width of a general tooltip in pixels.

hide

flag of tooltips displaying.

formats

map of columns to format string of its value.
